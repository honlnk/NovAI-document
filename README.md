# NovAI（诺艾）

> 一个以 RAG 两阶段检索为引擎、以文件系统为基石、以 Git 为安全网的长篇小说创作平台。

## 项目信息

- **文档仓库**：[github.com/honlnk/NovAI-document](https://github.com/honlnk/NovAI-document)
- **代码仓库**：待创建
- **项目主页**：暂定 `novai.honlnk.top`
- **技术栈**：Vue + 纯前端（无自建后端），后续 Tauri（桌面）+ Capacitor（移动）
- **支持平台**：第一阶段浏览器端，后续 Windows / macOS / Android
